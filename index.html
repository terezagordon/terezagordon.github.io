Project Idea

Overview

The proposed idea is to create a digital frame that will allow users to see photos they have taken on their phones, cameras, or other devices in a frame without having to print them. The user view one photo or a slideshow of photos which are stored on the cloud on the digital frame. The digital frame can be controlled by the users device to change the image to a different image.

Motivation

This project will be interesting because users do not have to print out all their photos. Using a digital frame streamlines the process of changing images, because it is very simple to add and remove photos from an album of photos. This saves more time than the traditional method where you had to take out the back of the frame and replace the photos.

Description

The user needs to upload their images of choice onto Dropbox, which needs to be configured with command line program Rclone to be able to download the images and sync them to the Raspberry Pi. Images can be uploaded to Dropbox from any device. When this is done, the images will cycle through a slideshow and loop multiples. Or, if the user chooses they can just use one image on the digital photo frame.

Tools and Technologies

An LCD screen is needed in order for the user to be able to see the images. A Raspberry Pi 3 is a small computer which will be what powers the digital photo frame. A power supply such as a micro-USB power adapter is needed to make sure the Raspberry Pi is charged. A keyboard, micro SD Card of at least 8GB, mouse and HDMI cable are all needed to operate the Raspberry Pi. The Raspberry Pi OS will need to be installed onto the SD card. Users will upload photos to Dropbox. Rclone is an open source command line program that will be syncing photos uploaded to Dropbox and the digital photo frame. A frame is needed to place the LCD screen in.

Skills Required

The Raspberry Pi OS is based on the Debian Linux, so some basic knowledge in needed. However, this should not be difficult as the information is quite accessible online. An old laptop screen or monitor can be used, however, if a new one is purchased it may be expensive to be a decent quality one. It will not be very feasible to find the Raspberry Pis as many are out of stock due, and they are also quite expensive to buy online. The frame, can be purchased, but if there are no frames that match it, a custom one will either need to be purchased or made. Software and programs such as Dropbox and Rclone are all free and easy to use. Micro SD cards, keyboards, mouses, and HDMI cables are all very inexpensive and easy to find in stores.

Outcome

If the project is successful, users will be able to easily see photos they have taken on a photo frame without the need to even print them out and frame them. Users will not have to constantly take out the SD card in order to add or remove photos on it to see different images displayed. Users will be able to frame animated images which would not be possible on a traditional photo frame.
